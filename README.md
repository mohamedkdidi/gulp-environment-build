# Gulp fron­tend work­flow automa­tion

* Com­piles SCSS to CSS, using a cache to make it faster.
* Auto-pre­fix­es CSS for the browsers we support.
* Ugli­fies JavaScripts.
* Han­dles live reloading.
* Compress the images.
* Zip the build.
* Clean dist folder.

## Installation

```sh
  $ npm install
```

Using yarn

```sh
  $ yarn install
```

## Run

```sh
  $ gulp
```


http://localhost:8080/
